# Spooky Season

## About
Spooky Season is a platformer where you are a ghost, and can switch between a physical form and one which can phase through matter...
## Controls
### IF THE DOOR DOESN'T WORK, JUMP AND TRY AGAIN
### Requires a keyboard
A and D to move left and right in normal mode
Space to jump in normal mode
Arrow keys to move in ghost mode
Shift to toggle modes
